# Starglow Modoki

After Effects の Starglow 風に、静止画の明るい部分から星形の光条とグローを作る小さなブラウザツールです。

## 使い方

1. `index.html` をブラウザで開く。
2. 画像をドロップするか、「画像を開く」から読み込む。
3. スライダーやプリセットで見た目を調整する。
4. 「PNG保存」で書き出す。

## 調整のコツ

- `しきい値` を上げると、本当に明るい部分だけが光ります。
- `長さ` と `強さ` を上げると、Starglow らしい光条が伸びます。
- `広がり` を上げると、柔らかいにじみが増えます。
- `元画像の色を混ぜる` を切ると、選んだ光の色が強く出ます。
